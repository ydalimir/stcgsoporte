# **App Name**: STICS Support Hub

## Core Features:

- Ticket Submission: Users can submit support tickets detailing their issues with home, commercial, or industrial equipment. They must specify the equipment type, a description, and urgency.
- Service Catalog Display: Showcase the range of services offered, including installation, maintenance, and repair of various equipment types.
- Ticket Prioritization: Analyze ticket descriptions and automatically assign a priority level (low, medium, high) based on keywords, urgency indicators, and equipment type.
- Admin Dashboard: A dedicated dashboard for administrators to view, manage, and assign support tickets. Admins can also update service catalogs and manage user accounts.
- Ticket Status Updates: Users receive automated notifications on ticket status changes (e.g., 'received,' 'in progress,' 'resolved') via email and within the app.
- Quotation Requests: Enable users to request quotations for specific services, which are then routed to administrators for review and response.
- FAQ and Help Resources: Provide a comprehensive FAQ section and help resources to assist users in resolving common issues independently.
- Firebase Integration: Integrate with Firebase for user authentication, data storage, and real-time updates.

## Style Guidelines:

- Primary color: A vibrant blue (#2E9AFE) to evoke trust, reliability, and professionalism.
- Background color: Light blue (#EBF5FF), providing a clean, airy backdrop for content.
- Accent color: A contrasting orange (#FF851B) to highlight calls to action, creating visual interest and guiding user interaction.
- Font pairing: 'Space Grotesk' sans-serif for headlines and short blocks of text, 'Inter' sans-serif for body text.
- Use clean, modern icons to represent service categories, ticket statuses, and equipment types. Aim for visual clarity and ease of understanding.
- Implement a responsive, user-friendly layout. Ensure readability, intuitive navigation, and ease of access across devices. Use card-based layouts to highlight individual services or FAQs.
- Use subtle transitions and loading animations to enhance the user experience. Animate ticket status changes, service catalog updates, and form submissions to provide clear feedback.
